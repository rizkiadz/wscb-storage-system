Setup Firebase
Ikuti langkah berikut untuk menghubungkan ke database Firebase:
1
Buka console.firebase.google.com → buat project baru
2
Klik Tambahkan aplikasi → ikon Web → daftarkan app → salin konfigurasi
3
Menu Build → Firestore Database → Create database → pilih test mode
4
Menu Build → Authentication → Get started → aktifkan Email/Password
5
Isi form di bawah dengan nilai dari konfigurasi Firebase Anda
Konfigurasi disimpan di browser Anda saja, tidak dikirim ke mana pun.
